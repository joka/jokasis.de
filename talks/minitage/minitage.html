minitage - install

  • Bootstrap Python

  • run virtualenv

  • easy_install

    • minitage.core

    • minitage.paste

minitage - new minibuild (package)

  • minibuild config file: MyProject-0.1

    • dependencies: xslt2, pil, python-2.6 ...

    • src_uri: to checkout your buildoutproject

  • "minimerge MyProject-0.1"

    • compiles all minibuild depencenies

    • runs your buildout.cfg

porting an existing buildout

  • python = ${buildout:directory}/../../dependencies/python-X.XX...

  • eggs-directory = ${buildout:directory}/../../eggs/cache

  • replace standard recipies:

    • zc.recipe.egg -> minitage.recipe.egg ...

    • or just use: extensions = buildout.minitagificator

install plone3 + varnish2 + postgresql

  • $paster create -t minitage.plone3 MyProject with_psycopg2=yes mode=relstorage

  • minimerge plony

  • $paster create -t minitage.instances.varnish2 MyProject

  • $paster create -t minitage.instances.postgresql MyProject
